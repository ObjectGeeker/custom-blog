# 博客系统

> 目前仅限于给本人搭建一个个人博客网站，由于目前市面上很多开源博客系统无法达到完全客制化的目的，所以想开发一个完全符合本人审美的博客系统



## 需求分析

### 用户端

1. 用户可在主页查看发布的文章
2. 当用户阅读到文章底部的时候会增加文章的阅读量
3. 用户可以通过文章标题、标签、分类进行文章搜索



### 管理员端

1. 管理员可通过账号密码登录管理员后台
2. 管理员可控制文章的标签，分类
3. 分类和标签都可以有多级，但最高不能超过四级
4. 管理员可以编写文章，最后都会渲染成markdown格式进行保存和展示
5. 管理员在编写文章时必须选分类和标签



### 页面原型

1. 整个页面一定要简洁，不需要花里胡哨的光效
2. 整个项目分为上下布局，头部就是左侧网站标题，右侧就是分类，通过下拉框展示一级分类，二级分类
3. 主页首先展示个人简历信息，下部分就是最新发布的五篇文章
4. 文章页面，左侧是当前分类下的其它文章，中间是文章内容，右侧是文章的一级标题，二级标题等
5. 在文章的最顶层，展示文章标题，编写时间，编写人，分类，标签等信息
6. 网站支持暗色模式切换



## 技术选型

### 后端

1. SpringBoot 3.5.13
2. SpringSecurity
3. MybatisPlus
4. Redis（用于对分类标签等频繁查询但不频繁变更的内容进行缓存）
5. MySQL（使用LONGTEXT字段存储文章内容）
6. 对象存储（存储图片信息）



### 前端

1. React + Next.js
2. MDX（Markdown渲染）
3. Tailwind CSS
